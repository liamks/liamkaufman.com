Adding Authentication, Waiting Lists and Sign Ups to an Express App Using Drawbridge.js and Redis

Apr 21st, 2012

There are several popular modules for adding password authenticate to an Express app. Unfortunately, they require creating lots of boilerplate code and views. Given the ubiquity of authentication I prefer the approach that authentication libraries like Devise take: they generate code and views, and you’re free to modify or delete what’s created. The advantage of that approach is it takes a few minutes to add authentication to an application. Given the options for Express I set about to create an authentication module that would make adding authentication extremely quick and easy. However, I also wanted the module to handle sign ups (the type you see on a just-launched startup’s page) and to handle waiting lists and invitations. Thus, I created Drawbridge.js.

Redis for Persistence

Drawbridge.js uses Redis to persists its data, but it’s possible for developers to create other database adapters for Drawbridge (pull-requests accepted). Drawbridge sends emails using either nodemailer, or the postmark modules. Likewise developers are free to add additional email adapters.

Drawbridge Screencast

I’ve created a short screencast to show how easy it is to add drawbridge to an existing application. Before you watch the screencast it’s important that I outline a couple of caveats:

  1. Drawbridge is not ready for production - it’s basically a working prototype.
  2. Drawbridge views and variables are inconsistently named, that will need to be fixed.
  3. The code needs refactoring and testing.
  4. Drawbridge needs to be picked apart for security issues.

With those caveats out of the way here is the video:

Drawbridge.js from Liam Kaufman on Vimeo.

While I plan on refactoring Drawbridge’s source code, I first wanted to get feedback from the community and see if there’s any interest before I invest too much time. If you’re interested in Drawbridge watch the project on Github and try and get it working on your toy Express apps. I welcome feedback on both the architecture of Drawbridge and its security.
